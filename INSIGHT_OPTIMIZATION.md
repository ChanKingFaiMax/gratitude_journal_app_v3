# 洞察功能全面优化方案

## 问题清单

### 1. 评论人与理论不匹配 ❌
**现状**：所有洞察类型都有副标题提到理论，但没有显示对应的专家评论人

**解决方案**：为每个洞察类型添加匹配的已逝世专家

| 洞察类型 | 理论基础 | 专家选择 | 理由 |
|---------|---------|---------|------|
| 我的人物关系 | 社会网络分析 | **Mark Granovetter** (马克·格兰诺维特，1943-，仍在世) | 社会网络分析先驱，"弱连接的力量"理论创始人 |
| 我的意识层级 | David Hawkins意识地图 | **David R. Hawkins** (大卫·霍金斯，1927-2012) | 意识层级地图创始人 |
| 我的成长 | David Hawkins意识层级 | **David R. Hawkins** (大卫·霍金斯，1927-2012) | 意识层级地图创始人 |
| 我近期可以注意的 | 正念觉察理论 | **Thích Nhất Hạnh** (一行禅师，1926-2022) | 正念大师，将正念带入西方 |
| 如何梳理我的内在矛盾 | 荣格心理学 | **Carl Jung** (卡尔·荣格，1875-1961) | 分析心理学创始人 |

**注意**：Mark Granovetter仍在世，需要替换为已逝世的社会学家

**替代方案**：
- **Georg Simmel** (格奥尔格·齐美尔，1858-1918) - 社会学奠基人，研究社会关系和互动
- **Émile Durkheim** (埃米尔·涂尔干，1858-1917) - 社会学三大奠基人之一，研究社会连带

**最终选择**：**Georg Simmel** - 更符合"人物关系"主题

### 2. "亲爱的光之存有"等confusing称呼 ❌
**现状**：AI在生成"我的成长"分析时，使用了"亲爱的光之存有"这类新时代灵性术语

**解决方案**：
1. 在prompt中明确禁止使用新时代/灵性术语
2. 要求使用简单、直接的称呼（如"你"、"朋友"）
3. 避免过度神秘化的语言

**需要禁止的术语**：
- 光之存有 / Light Being
- 星际种子 / Starseed  
- 扬升 / Ascension
- 水晶儿童 / Crystal Children
- 神圣几何 / Sacred Geometry
- 第三眼 / Third Eye
- 脉轮 / Chakra（除非明确相关）

### 3. 缺少评论人显示 ❌
**现状**：前端没有显示评论人的名字、头像或签名

**解决方案**：
1. 在每个洞察结果底部添加"评论人"卡片
2. 显示专家名字、生卒年份、简短介绍
3. 使用emoji作为头像（避免版权问题）

**UI设计**：
```
┌─────────────────────────────────────┐
│  [洞察内容]                         │
│                                     │
│  ─────────────────────────────     │
│                                     │
│  💭 来自 Carl Jung 的视角           │
│  卡尔·荣格 (1875-1961)              │
│  分析心理学创始人                   │
└─────────────────────────────────────┘
```

### 4. Prompt优化需求
**需要添加的约束**：
1. 禁止使用新时代/灵性术语
2. 使用简单、直接的称呼
3. 保持专业但温暖的语气
4. 避免过度神秘化
5. 基于科学/心理学框架，而非玄学
