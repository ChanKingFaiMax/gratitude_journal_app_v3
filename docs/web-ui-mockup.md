## 7. 交互说明

### 7.1 快捷键

| 快捷键 | 功能 |
|-------|------|
| `Cmd/Ctrl + S` | 保存当前日记 |
| `Cmd/Ctrl + Enter` | 完成并保存 |
| `Cmd/Ctrl + I` | 获取智者启示 |
| `Esc` | 返回上一页 |

### 7.2 动画效果

| 元素 | 动画 | 时长 |
|-----|------|------|
| 智者启示出现 | 从右侧滑入 + 淡入 | 300ms |
| 卡片切换 | 左右滑动 | 250ms |
| 按钮点击 | 缩放 0.97 | 100ms |
| 页面切换 | 淡入淡出 | 200ms |
