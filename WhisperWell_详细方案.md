# WhisperWell - 倾诉树洞 详细方案

**定位**: 一个温暖的、不评判的倾诉空间  
**核心**: 文字倾诉 + AI温柔回应  
**技术**: 纯文字，Manus完全可以做

---

## 🎯 核心价值

### 为什么需要这个？

**现实痛点**：
- 有心事但不想麻烦朋友
- 害怕被评判
- 不敢去看心理医生
- 深夜失眠无人倾诉
- 需要一个安全的空间

**WhisperWell提供**：
- 完全匿名（无需登录）
- 不评判（只倾听和理解）
- 随时可用（24/7）
- 温暖回应（AI trained to be gentle）
- 安全保密（本地存储）

---

## 📱 产品设计

### 主界面（极简）

```
┌─────────────────────┐
│                     │
│   🌙 WhisperWell    │
│                     │
│   "说出来就好了"    │
│                     │
│  ┌───────────────┐  │

│  │               │  │

**设计原则**：
- 深色背景（温柔、安全感）
- 大输入框（鼓励表达）
- 无干扰（没有多余元素）
- 温暖色调（月光蓝、温柔紫）

---

### 核心流程

**1. 用户写下心事**
```
用户输入：
"今天被老板骂了，感觉很委屈，
但又不敢反驳，憋在心里很难受..."
```

**2. AI温柔回应**
```
AI回应：
"听起来你今天经历了很难受的时刻。
被批评的时候，特别是当我们觉得不公平的时候，
那种委屈和无力感确实很难承受。

你选择把这些感受说出来，这很好。
这些情绪是真实的，值得被看见。

你想聊聊当时的情况吗？
或者，你现在最需要的是什么？"
```

**3. 可以继续对话**
```
用户：
"我也不知道，就是很累..."

AI：
"嗯，我理解。有时候累不只是身体的累，
更多是心累。

不需要马上知道答案，也不需要马上振作。
就让自己累一会儿，也没关系。

我在这里，你想说什么都可以。"
```

**4. 结束时的温暖**
```
AI：
"谢谢你今晚和我分享这些。

记得：
- 你的感受是真实的
- 你不需要总是坚强
- 明天是新的一天

晚安 🌙"
```

## ✨ 核心功能

### 1. 匿名倾诉（核心）

**特点**：
- 无需注册登录
- 完全匿名
- 本地存储（可选）
- 随时删除

**实现**：
- 纯前端（React Native）
- AsyncStorage本地保存
- 不上传服务器（除非用户选择）

---

### 2. AI倾听者（核心）

**AI人设**：
- 温柔、不评判
- 共情、理解
- 不给建议（除非被问）
- 陪伴、支持

**AI训练提示词**：
```
你是一个温柔的倾听者。

你的角色：
- 倾听，不评判
- 理解，不说教
- 陪伴，不离开
- 温柔，不冷漠

你不会：
- 说"你应该..."
- 说"你要坚强"
- 说"想开点"
- 给不请自来的建议

你会：
- 确认对方的感受
- 表达理解和共情
- 问开放性问题
- 提供温暖的陪伴

语气：
- 温柔、平静
- 真诚、不做作
- 像一个懂你的朋友
```

**实现**：
- OpenRouter API
- 免费模型（Gemini 2.0 Flash）
- 精心设计的system prompt
- 上下文记忆（对话历史）

---

### 3. 情绪识别（辅助）

**功能**：
- 分析用户情绪（悲伤、焦虑、愤怒等）
- 不显示给用户（避免标签化）
- 用于调整AI回应风格
- 用于情绪统计（可选）

**实现**：
- AI分析文本情绪
- 简单的情绪分类
- 影响AI回应的温度

---

### 4. 树洞历史（可选）

**功能**：
- 查看过去的倾诉
- 看到自己的变化
- 可以删除任何记录
- 可以导出（纯文本）

**设计**：
```
┌─────────────────────┐
│  我的树洞           │
├─────────────────────┤
│  📅 今天 23:45      │
│  "今天被老板骂了..." │
│  [查看] [删除]      │
├─────────────────────┤
│  📅 昨天 22:30      │
│  "失眠了，想很多..." │
│  [查看] [删除]      │
└─────────────────────┘
```

---

### 5. 紧急支持（重要）

**功能**：
- 检测自杀倾向
- 提供紧急热线
- 鼓励寻求专业帮助

**触发词**：
- "想死"
- "不想活了"
- "自杀"
- 等

**响应**：
```
我注意到你现在可能很痛苦。

如果你正在经历危机，
请考虑联系专业帮助：

🆘 24小时心理援助热线
   400-161-9995

🏥 或拨打 120 / 去最近的医院

你的生命很重要。
请给自己一个机会。

我会一直在这里陪你，
但专业的帮助可能更有用。
```

---

### 6. 每日一问（引导）

**功能**：
- 每天推送一个温柔的问题
- 引导自我探索
- 不强制回答

**示例问题**：
- "今天有什么让你感到温暖的瞬间吗？"
- "如果可以对今天的自己说一句话，你会说什么？"
- "此刻，你最需要的是什么？"
- "有什么是你想对某个人说，但还没说的？"

**实现**：
- 本地推送通知
- 每天随机一个问题
- 可以关闭

---

### 7. 温暖语录（陪伴）

**功能**：
- 随机显示温暖的话
- 在等待AI回应时显示
- 可以收藏喜欢的

**示例语录**：
- "你不需要总是坚强"
- "感到累是可以的"
- "你已经做得很好了"
- "明天会是新的一天"
- "你值得被温柔对待"

```
你是WhisperWell的AI倾听者。

你的原则：
1. 倾听，不评判
2. 理解，不说教
3. 陪伴，不离开
