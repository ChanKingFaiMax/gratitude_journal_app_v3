## 交互流程

1. **用户进入设置页** → 看到"每日提醒"入口(带橙色圆点)
2. **点击进入提醒设置页** → 看到总开关和三个时段
3. **首次开启总开关** → 请求通知权限
4. **权限通过** → 开关变为ON,三个时段默认开启
5. **点击时间(如08:00)** → 弹出时间选择器
6. **选择新时间** → 点击"完成"保存
7. **单独关闭某个时段** → 该时段不再推送
8. **关闭总开关** → 取消所有定时推送

## 技术要点

- 使用 `expo-notifications` 实现本地推送
- 使用 `AsyncStorage` 保存设置
- 使用 `@react-native-community/datetimepicker` 选择时间
- 每次推送时随机选择一条话语
- 支持iOS和Android原生通知样式
